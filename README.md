一、本仓库实现的改进
1. 逐通道自适应收缩阈值
原始 BDAT 使用一个所有通道共享的标量平衡参数。这里将该参数扩展为每个通道各自独立学习的向量(torch.ones(64) * 0.48)，使高频通道和低频通道能够学习各自独立的降噪行为。
2. 可学习的三流融合权重
不再用固定等权重相加的方式融合三条流的输出，而是通过一个经过 softmax 归一化的可学习权重向量进行融合，使模型能够在训练过程中自适应地重新调整粗粒度、细粒度、降噪三条流的相对重要性。
3. 中心损失(Center Loss)正则化
增加了一个辅助的中心损失项(L_total = L_CE + λ · L_center，λ = 0.01)，用于约束同类特征向各自的类中心靠拢，类中心参数与网络参数一起联合优化。
4. 课程式加噪训练策略
不再在整个训练过程中固定注入同一噪声水平，而是分三个阶段调度噪声：热身阶段(SNR = 6 dB)、主训练阶段(SNR = 0 dB)，以及最后阶段(从 {6, 4, 2, 0, −2, −4} dB 中随机采样)，旨在提升模型在不同噪声条件下的鲁棒性。

二、数据集
•	CWRU 轴承数据集(凯斯西储大学)，驱动端加速度计信号，采样频率 48 kHz，1 马力电机负载。
•	四种健康状态：正常(N)、滚动体故障(B)、内圈故障(IR)、外圈故障(OR)。
•	原始信号被切分为长度 1024 的窗口(不重叠或部分重叠)，并通过 Z-score 标准化。
•	由于文件较大，数据文件未包含在本仓库中，请从 CWRU 轴承数据中心下载。

四、使用方法
1.	将 notebook 中的 base_path 修改为你本地 CWRU .mat 文件所在路径。
2.	依次运行各代码单元，完成以下步骤： 
o	加载并可视化原始振动信号
o	构建滑窗数据集和 dataloader(可选注入高斯噪声)
o	训练 EWSNet 模型(基础版或课程加噪版)
o	在 −6 dB 到 10 dB 的多个信噪比水平下评估噪声鲁棒性
o	生成可视化结果：训练曲线、混淆矩阵、t-SNE 特征嵌入、小波核演化、频率激活映射(FAM)
